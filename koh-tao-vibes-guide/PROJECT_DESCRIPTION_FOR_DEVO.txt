PROJECT DESCRIPTION FOR DEVO.DEV - KOH TAO GUIDE

PROJECT OVERVIEW:
Create a comprehensive travel guide website for Koh Tao, Thailand - a tropical diving paradise. This is a modern, luxury-themed single-page application that serves as the ultimate travel companion for visitors to explore dive sites, beaches, destinations, activities, accommodations, and local businesses.

TECHNOLOGY STACK:
- React 18.3.1 with TypeScript
- Vite as build tool
- React Router DOM for routing
- Tailwind CSS for styling
- shadcn/ui component library
- TanStack React Query for data management
- Lucide React for icons
- Context API for state management (Language switching)

DESIGN SYSTEM - LUXURY DARK THEME:
The entire website uses a sophisticated dark luxury theme with the following color palette:
- Background: Deep dark navy (HSL: 220 20% 8%)
- Foreground: Light cream/white (HSL: 180 30% 98%)
- Primary: Deep ocean teal (HSL: 180 70% 45%)
- Secondary: Rich gold/amber (HSL: 45 85% 55%)
- Accent: Coral/pink (HSL: 15 75% 60%)
- Cards: Dark navy (HSL: 220 25% 12%)
- Borders: Subtle dark borders (HSL: 220 25% 20%)

All gradients use teal-to-gold combinations. The design features:
- Smooth animations and transitions
- Backdrop blur effects
- Luxury shadows and glows
- Rounded corners (0.75rem radius)
- Premium glassmorphism effects

COMPONENT FEATURES:

INTERACTIVITY:
- Smooth scroll navigation to sections
- Toast notifications for user actions (booking, searching, etc.)
- Language switching (English, Spanish, Thai) with localStorage persistence
- Search functionality in hero section
- Hover effects on all cards (scale, shadow, border color changes)
- Mobile-responsive design with hamburger menu
- All buttons have gradient backgrounds and hover effects

DATA STRUCTURE:
The website uses JSON data files for:
- destinations.json: Beach and viewpoint locations with images, descriptions, activities
- activities.json: Activity listings with prices, features, icons
- accommodations.json: Hotel/resort listings with ratings, prices, amenities, locations
- businesses.json: Business directory with categories, ratings, hours, contact info

LANGUAGE SUPPORT:
Full internationalization with 4 languages:
- English (default)
- Spanish (Español)
- Thai (ไทย)
- German (Deutsch)

All text content is translatable including:
- Navigation items
- Section titles and descriptions
- Button labels
- Toast messages

UI COMPONENTS USED:
